# Procedimiento para el Balanceo de Camiones y Trailers

## Título
Procedimiento para el Balanceo de Llantas (Morenas) de Camiones y Trailers

## Servicios Principales
El proceso de balanceo de camiones se realiza en varias etapas para garantizar que todos los componentes estén ajustados correctamente:

1. **Inspección inicial de los neumáticos**: El primer paso es inspeccionar los neumáticos para identificar cualquier signo de desgaste irregular o daño. Esto ayuda a determinar si el balanceo es necesario y si los neumáticos deben ser reemplazados.

2. **Desmontaje de las ruedas**: Para realizar el balanceo, es necesario retirar las ruedas del camión. Esto permite al mecánico examinar los neumáticos y los aros en detalle.

3. **Balanceo estático y dinámico**: El balanceo se realiza utilizando máquinas especializadas. En el balanceo estático, se ajustan los pesos para equilibrar las ruedas. El balanceo dinámico, por otro lado, evalúa las fuerzas que se generan cuando el camión está en movimiento, permitiendo un ajuste más preciso.

4. **Montaje y prueba de manejo**: Una vez que se ha realizado el balanceo, las ruedas se vuelven a montar y se realiza una prueba de manejo para asegurar que el camión se maneja de manera estable y sin vibraciones.
